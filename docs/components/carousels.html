      Carousels

      To create a new carousel, invoke either the loopedCarousel(el, opts) or finiteCarousel(el, opts) method. These methods take 2 arguments:

      Parameter Description
      el id or element of the slider element (must have the classes ‘carousel & carousel–looped’)
      opts The ‘slick’ slider options to pass to the carousel (see here)

      In the ‘opts’ object, you must not override the dots option, this value of this must remain false
